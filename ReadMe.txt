Gradforce is an NZ-based recruiting agency of which had a need for a web-based portal for applicants to apply for jobs. This program is HTML, CSS, Javascript, MYSQL & PHP.
It includes many features including login authentication from a database to the user which allows all users to have a unique login. From logging, listed jobs provided by the recruiter will be available to view for the user. 
A user will then be able to see the job listed and read descriptions. From there,the user can then apply for the job by entering in required fields and uploading a resume.
On the admin's side, there are designated logins for individual admins. From the admin dashboard page, an admin can create new job listings.
These listings will instantly appear on the user site dashboard once approved.
